# SQL Operations KPI Analysis

## Business Problem
Operations teams need visibility into SLA adherence, turnaround time (TAT), and agent performance.

## Dataset
- Orders and agents data (simulated)
- Focus on delivery timelines and delays

## SQL Concepts Used
- Joins
- CTEs
- Aggregations
- Date functions
- Filters and ranking

## Key Insights
- Identified cities with higher delivery TAT
- Calculated SLA breach percentage by month
- Ranked agents by delayed orders

## How to Run
1. Execute schema.sql
2. Load sample_data.sql
3. Run queries.sql
